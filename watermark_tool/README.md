画像にウォーターマークやノイズを追加し、AI学習への利用を妨害することでクリエイターの権利を保護する画像保護ツールです。

## 1. 開発背景
生成AIによる無断学習からイラスト作品を守るため、視覚的な著作権表示（ウォーターマーク）と、AIの学習効率を低下させるノイズ（Mist）およびメタデータ汚染（Poisoning）を同時に実行できるツールとして開発しました。

## 2. 主な機能

### AI阻害機能
* ミスト強度（ノイズ）: 画像全体にランダムノイズを重畳。値が大きいほどノイズを強化し、AIによる特徴量抽出を阻害します。
* メタデータ汚染（Poisoning）: 保存時にPNGメタデータへAI学習用のネガティブタグ（`bad anatomy`, `low quality`等）を埋め込みます。
* ノイズ確認モード: 元画像との差分のみを強調表示し、保護処理による画像の変化を視覚的に確認できます。

### ウォーターマーク設定
* 自由配置モード: プレビュー上を直接クリックまたはドラッグして、直感的に配置位置を指定可能です。
* タイル配置モード: 画像全体に斜め方向で繰り返し配置し、部分的な切り抜きによる無断転載を防止します。
* 詳細カスタマイズ: テキスト内容、サイズ、透明度、太さ、色、フォントを自由に設定でき、プリセットとして保存可能です。

### 解析・確認機能
* メタデータ検査: 指定した画像のメタデータを解析します。AI学習阻害用のタグが含まれる場合は、`[!! POISON !!]` ラベルを付与して表示し、保護状態を一目で確認できます。

## 3. 技術スタック
* 言語: Python 3.9+
* GUI: tkinter (Python標準ライブラリ)
* 画像処理: Pillow (PIL)
* 設定管理: JSON

## 4. 実行方法

1. ライブラリのインストール:
   ```bash
   pip install pillow
   ```

2. ツールの起動:
   ```bash
   python watermark_tool/water_noize.py
   ```

## 5. ファイル構成

- `README.md` (本ファイル)

- `watermark_tool/`
  - `water_noize.py` : メインプログラム
  - `watermark_settings.json` : 設定保存用（初回起動時に自動生成）

## 6. 注意事項

- ファイル形式: 保存およびメタデータ処理は PNG形式のみ 対応しています。

- フォント: フォント未指定時はシステム標準フォント（Windows: Arial等）が適用されます。OS環境によりデフォルトのフォントスタイルが異なる場合があります。

- パフォーマンス: 高解像度の画像では、リアルタイムプレビューの更新に時間を要する場合があります。

## 7. 汚染用埋め込みタグ一覧

埋め込まれるタグの内容：

```text
lowres, bad anatomy, bad hands, text, error, missing fingers, extra digit, fewer digits, cropped, worst quality, low quality, normal quality, jpeg artifacts, signature, watermark, username, blurry
```
